# SOS Jukebox Prototype

Static clickable prototype for the SOS / Sing One Song jukebox flow.

- `index.html` is the v5 shell/home UI.
- `v4/` is the embedded v4 clickable-plumbing bridge used by Jukebox, Arcade, and Drills.
- GitHub Pages deploys automatically on every push to `main` via `.github/workflows/pages.yml`.

Public URL after the first successful Pages deploy:

https://senapaty.github.io/sos-jukebox/
